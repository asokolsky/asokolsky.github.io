# My Emacs Customization

I use

* json-mode
* yaml-mode
* markdown-mode:
```
M-x package-install <RET> markdown-mode <RET>
```
* [dracula-theme](https://draculatheme.com/emacs):
```
M-x package-install <RET> dracula-theme <RET>
```
