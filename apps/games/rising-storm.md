# Rising Storm/Red Orchestra 2 Multiplayer

## Change resolution

### via console

1. Open the console with `~` or `F3`
1. Type in the following: `setres 3440x1440` or `setres 3840x1600`
1. Press Enter.

### via ini file

1. edit
   `Documents\My Games\RedOrchestra2\ROGame\Config\ROEngine.ini`;
1. change the values for `ResX` and `ResY` to match those of your monitor;
1. Save the changes made.

## Lunux Settings

Steam settings:

- command line: `PROTON_USE_WINED3D=1 %command% -nostartupmovies`
