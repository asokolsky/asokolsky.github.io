# K9s

Sources:

* [k9scli.io](https://k9scli.io/)

## Installation

On mac:
```sh
brew install k9s
```
On Linux (if you do not mind an old version):
```sh
sudo snap install k9s
```

## Important directories

Run `k9s info` for location of important directories, e.g. logs.
