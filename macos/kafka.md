# Using Confluent Cloud CLI

## Install on laptop

MacOS install:

```sh
brew install confluentinc/tap/cli
```

Verify:
```
> confluent version
confluent - Confluent CLI

Version:     v4.17.0
Git Ref:     4240da3d
Build Date:  2025-02-04T22:30:37Z
Go Version:  go1.22.7 (darwin/amd64)
Development: false
```
